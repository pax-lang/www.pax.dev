Pax is a cross-platform, highly native rendering engine & declarative user interface language.

Declare Pax UIs for native apps & websites, then drive logic with your favorite programming language â€” starting with Rust.

Features:

  • Fast (240FPS) rendering and a lightweight (<100KB) runtime
  • Native UI elements, text, and scrolling, with robust accessibility & screen reader support.
  • Creative freedom: mix & match drawing elements and UI elements in the same coordinate space
  • Reusable & composable UI components
  • Responsive 2D layouts and escape hatches for platform-specific customization
  • Powerful animation APIs
Pax is MIT & Apache licensed, being developed in the open. Currently pre-alpha.
