Pax is a declarative user interface language and 2D layout engine for Rust.

Build cross-platform apps, websites, and games, with:
  • Native UI elements, text, and scrolling
  • Responsive layouts
  • Reusable & composable UI components
  • Declarative templates & property bindings
  • Accessibility & screen reader support
  • Free-form vector drawing APIs
  • Powerful animation APIs
Pax is MIT & Apache licensed, being developed in the open in a pre-alpha public preview.
